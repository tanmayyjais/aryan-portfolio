# Aryan Kumar Cinematography Portfolio

An immersive cinematography portfolio built with Next.js, TypeScript, Tailwind CSS, Framer Motion, and Lenis smooth scrolling.

## Quick Start

1. Install dependencies:

```bash
npm install
```

2. Start the dev server:

```bash
npm run dev
```

3. Open `http://localhost:3000`

## Replacing Placeholder Content

- Edit portfolio data in `lib/site-data.ts`
- Replace section copy inside the components in `components/`
- Swap placeholder image URLs with your own assets in `public/` or remote image URLs
- Update metadata in `app/layout.tsx`
- Replace contact links in `components/contact-section.tsx`

## Suggested Asset Folders

Create these folders when you are ready to add real media:

- `public/images/portraits`
- `public/images/projects`
- `public/images/gallery`
- `public/videos`

Then switch image URLs to local paths such as `/images/projects/ember-mile-cover.jpg`.
